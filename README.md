# Modo Var Web 🟢

## Overview

**Modo Var** is a beta-stage online store dedicated to selling football jerseys. This project is being developed with the goal of becoming a fully functional and real-world e-commerce platform.

The website is currently in **beta version**, meaning new features, optimizations, and structural improvements are continuously being implemented.

---

## 🎯 Project Purpose

The main objective of this project is to build a professional online store that provides:

- A modern and responsive user interface
- Smooth navigation experience
- Product catalog with detailed information
- Secure checkout system (future implementation)
- Newsletter and customer interaction tools
- Scalable architecture for production deployment

---

## ⚽ About Modo Var

**Modo Var** is an emerging digital store focused on football jerseys and sports apparel. The platform is designed to simulate a real commercial environment and evolve into a fully operational marketplace.
